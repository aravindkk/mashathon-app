About

Cool For Me is a state of the art prediction system that assesses accurately if a place will suit you or not. Using advanced machine learning technologies such as Face Detection, Sentiment Analysis and data about Air Quality from Breezometer API, this app whips up the sane message that you are looking for.

About

Cool For Me is a state of the art prediction system that assesses accurately if a place will suit you or not. Using advanced machine learning technologies such as Face Detection(which evaluates your age and immune system levels), Sentiment Analysis(which evaluates the air quality recommendations based on age group) and data about Air Quality from Breezometer API, this app whips up the sane message that you are looking for.
